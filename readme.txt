Task 1: Read a File and Handle Errors
Functionality:

>>Opens a file named sample.txt.

>>Reads the file line by line, printing each line with its corresponding line number.

>>If the file does not exist, it catches the error (FileNotFoundError) and displays a friendly error message like:
"Error: The file 'sample.txt' was not found."


Task 2: Write and Append Data to a File
Functionality:

>>Takes input from the user and writes it to a file named output.txt.

>This step creates or overwrites the file.

>>Then it takes another input and appends that to the same file.

>This step preserves existing content and adds new data.

>>Finally, it reads the entire file and displays its final content to the user.
